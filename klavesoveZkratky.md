# Jak psát některé znaky na klávesnici (české, pokud neuvedeno jinak)

##### Vlnovka ~

```
Alt Gr + Shift + 1 (nahoře nad písmeny, ne v numerické klávesnici)
```

##### Hashtag '#'

```
Alt Gr + X
```

##### Ampersand &

```
Alt Gr + C
```

##### "Trubka" |

```
Alt Gr + W
```

Vítejte v systému správy uživatelů

Tento systém umožňuje správu uživatelů s využitím Flask frameworku, SQLAlchemy ORM a Alembic migrací.

Dostupné funkce:

  • Zobrazení seznamu všech uživatelů
  • Přidání nového uživatele
  • Editace existujícího uživatele
  • Mazání uživatelů
  • REST API pro všechny operace
Zobrazit uživatele Přidat uživatele

API Endpointy:

  • GET /api/users - Seznam všech uživatelů
  • POST /api/users - Vytvoření nového uživatele
  • GET /api/users/<id> - Detail uživatele
  • PUT /api/users/<id> - Aktualizace uživatele
  • DELETE /api/users/<id> - Smazání uživatele
